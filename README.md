# Pyramid Text

## Objective
Create a pyramid of text using user input.

## Motivation
I saw YouTube comments making pyramids out of text and thought there should be an easier way to make those comments.

## Example
Input: 

`HelloWorld`

Output:

```

H
He
Hel
Hell
Hello
HelloW
HelloWo
HelloWor
HelloWorl
HelloWorld
HelloWorl
HelloWor
HelloWo
HelloW
Hello
Hell
Hel
He
H

```
